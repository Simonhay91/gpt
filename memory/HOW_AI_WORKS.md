# Planet Knowledge - Как работает AI с источниками

## Простое объяснение

### Что такое "Источник" (Source)?
Источник - это файл или ссылка, которую вы загружаете в систему. AI использует эти источники чтобы отвечать на ваши вопросы.

**Примеры источников:**
- PDF документ с инструкцией
- Excel таблица с продуктами
- Word файл с политикой компании
- Ссылка на веб-страницу

---

## Как AI "читает" ваши документы

### Шаг 1: Загрузка файла
```
Вы загружаете: "Каталог_продуктов.xlsx"
                    ↓
Система сохраняет файл и начинает обработку
```

### Шаг 2: Разбиение на части (Chunking)
AI не может прочитать весь документ сразу. Система разбивает его на маленькие части:

```
┌─────────────────────────────────────────────────────┐
│           Каталог_продуктов.xlsx (3000 строк)       │
└─────────────────────────────────────────────────────┘
                         ↓
┌─────────┐ ┌─────────┐ ┌─────────┐ ┌─────────┐
│ Chunk 1 │ │ Chunk 2 │ │ Chunk 3 │ │ Chunk N │
│ Строки  │ │ Строки  │ │ Строки  │ │   ...   │
│  1-50   │ │ 51-100  │ │ 101-150 │ │         │
└─────────┘ └─────────┘ └─────────┘ └─────────┘
```

**Для Excel/CSV:** Каждая строка становится отдельным chunk с заголовками колонок:
```
"Название: Оптический кабель AS-L-16FO, Код: 20007, Цена: 521, Категория: Кабели"
```

### Шаг 3: Создание "отпечатков" (Embeddings)
Каждый chunk превращается в числовой вектор - как уникальный отпечаток пальца:

```
"Оптический кабель AS-L-16FO..."  →  [0.23, -0.45, 0.12, 0.87, ...]
"Патч-корд LC/LC дуплекс..."      →  [0.15, -0.33, 0.45, 0.22, ...]
```

Эти "отпечатки" хранятся в базе данных.

## Как AI отвечает на вопросы

### Шаг 1: Вы задаёте вопрос
```
"Покажи все оптические кабели с ценой больше 500"
```

### Шаг 2: Поиск похожих частей
Ваш вопрос тоже превращается в "отпечаток" и система ищет похожие chunks:

```
Ваш вопрос: [0.21, -0.42, 0.15, 0.85, ...]
                         ↓
            Поиск похожих в базе данных
                         ↓
┌────────────────────────────────────────────────────────────┐
│  Найдено 5 похожих chunks:                                 │
│  • Chunk 45: "Оптический кабель AS-L-16FO, Цена: 521..."  │
│  • Chunk 67: "Оптический кабель AS-L-24FO, Цена: 680..."  │
│  • Chunk 89: "Оптический кабель AS-L-8FO, Цена: 450..."   │
│  • Chunk 112: "Патч-корд оптический, Цена: 320..."        │
│  • Chunk 156: "Кабель ADSS-16, Цена: 590..."              │
└────────────────────────────────────────────────────────────┘
```

### Шаг 3: AI формирует ответ
GPT получает ваш вопрос + найденные части документов:

```
┌─────────────────────────────────────────────────────────────┐
│  Системный промпт: "Ты помощник. Отвечай на основе данных" │
├─────────────────────────────────────────────────────────────┤
│  Контекст из документов:                                    │
│  - Chunk 45: "Оптический кабель AS-L-16FO, Цена: 521..."   │
│  - Chunk 67: "Оптический кабель AS-L-24FO, Цена: 680..."   │
│  - ...                                                      │
├─────────────────────────────────────────────────────────────┤
│  Вопрос: "Покажи все оптические кабели с ценой > 500"      │
└─────────────────────────────────────────────────────────────┘
                              ↓
                           GPT-4.1
                              ↓
┌─────────────────────────────────────────────────────────────┐
│  Ответ:                                                     │
│  "Найдено 3 оптических кабеля с ценой больше 500:          │
│   1. AS-L-16FO - 521 руб.                                   │
│   2. AS-L-24FO - 680 руб.                                   │
│   3. ADSS-16 - 590 руб."                                    │
└─────────────────────────────────────────────────────────────┘
```

## Иерархия источников

### 4 уровня доступа к знаниям:

```
┌─────────────────────────────────────────────────────────────┐
│  🌍 ГЛОБАЛЬНЫЕ ИСТОЧНИКИ                                    │
│     Доступны ВСЕМ пользователям                             │
│     Пример: Общие политики компании, справочники            │
│     Управляет: Администратор                                │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│  🏢 ИСТОЧНИКИ ОТДЕЛА                                        │
│     Доступны сотрудникам отдела                             │
│     Пример: Документация отдела закупок                     │
│     Управляет: Менеджер отдела                              │
│     Требуют одобрения перед публикацией                     │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│  📁 ИСТОЧНИКИ ПРОЕКТА                                       │
│     Доступны участникам проекта                             │
│     Пример: ТЗ проекта, спецификации                        │
│     Управляет: Владелец проекта                             │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│  👤 ЛИЧНЫЕ ИСТОЧНИКИ                                        │
│     Доступны только вам                                     │
│     Пример: Ваши заметки, черновики                         │
│     Управляете: Вы                                          │
└─────────────────────────────────────────────────────────────┘
```

## Активные источники в чате

### Важно! AI использует только АКТИВНЫЕ источники

В каждом чате вы можете выбрать какие источники использовать:

```
┌─────────────────────────────────────────┐
│  Активные источники для этого чата:     │
│                                         │
│  ☑ Каталог_продуктов.xlsx              │
│  ☑ Прайс_лист_2024.pdf                 │
│  ☐ Инструкция_по_монтажу.docx          │
│  ☐ Контакты_поставщиков.csv            │
└─────────────────────────────────────────┘
```

**Если источник не активен** - AI его не увидит и не сможет использовать!

### Пример:
```
Вопрос: "Какой телефон у поставщика Оптоком?"

❌ Если "Контакты_поставщиков.csv" НЕ активен:
   → "У меня нет информации о контактах поставщиков.
      Пожалуйста, активируйте нужный источник."

✅ Если "Контакты_поставщиков.csv" активен:
   → "Телефон поставщика Оптоком: +7 (495) 123-45-67"
```

## Приоритет источников

Когда информация есть в нескольких источниках:

```
Приоритет (от высокого к низкому):

1️⃣ Источники ПРОЕКТА        ← Самый высокий приоритет
2️⃣ Источники ОТДЕЛА
3️⃣ ГЛОБАЛЬНЫЕ источники     ← Самый низкий приоритет
```

**Пример конфликта:**
```
Глобальный источник: "Цена кабеля AS-L-16FO: 500 руб."
Источник проекта:    "Цена кабеля AS-L-16FO: 521 руб."

AI ответит: "521 руб." (данные из проекта приоритетнее)
```

## Workflow одобрения (для источников отдела)

```
┌──────────┐      ┌────────────┐      ┌──────────┐
│  Черновик │ ──▶ │  На        │ ──▶ │ Активен  │
│  (Draft)  │     │  проверке  │     │ (Active) │
└──────────┘      └────────────┘      └──────────┘
     │                  │
     │                  ▼
     │            ┌──────────┐
     └──────────▶ │ Отклонён │
                  │(Rejected)│
                  └──────────┘
```

1. **Сотрудник** загружает документ → статус "Черновик"
2. **Сотрудник** отправляет на проверку → статус "На проверке"
3. **Менеджер** одобряет или отклоняет
4. Только **активные** источники используются AI

## Семантический кэш

Система запоминает ответы на похожие вопросы:

```
Первый вопрос:  "Какая цена на кабель AS-L-16FO?"
                          ↓
                    Запрос к GPT (5 сек)
                          ↓
                    Ответ: "521 руб."
                          ↓
                    💾 Сохранено в кэш

Похожий вопрос: "Сколько стоит кабель AS-L-16FO?"
                          ↓
                    Найдено в кэше! (0.1 сек)
                          ↓
                    Ответ: "521 руб. 📦"
                    (иконка означает ответ из кэша)
```

**Преимущества:**
- ⚡ Быстрее (мгновенный ответ)
- 💰 Дешевле (не тратит токены GPT)

## Excel/CSV Analyzer - отдельный инструмент

Для глубокого анализа таблиц есть отдельный инструмент:

```
┌─────────────────────────────────────────────────────────────┐
│  📊 Excel Analyzer                                          │
│                                                             │
│  Загрузите файл → Задавайте вопросы                        │
│                                                             │
│  "Покажи все продукты категории Optical cable"             │
│  "Найди дубликаты по коду"                                 │
│  "Посчитай сумму по колонке Price"                         │
│                                                             │
│  ⚠️ Ограничение: ~500 строк из-за лимита GPT               │
└─────────────────────────────────────────────────────────────┘
```

**Отличие от обычного чата:**
| | Обычный чат | Excel Analyzer |
|--|-------------|----------------|
| Файл | Chunked + векторный поиск | Весь файл в контекст |
| Поиск | Семантический (по смыслу) | Точный (все данные) |
| Лимит | Зависит от релевантности | ~500 строк |
| Для чего | Общие вопросы | Детальный анализ |

## Типичные проблемы и решения

### ❌ "AI не видит мой документ"
**Решение:** Проверьте что источник **активирован** в чате

### ❌ "AI даёт неполный ответ"
**Причина:** Найдено мало релевантных chunks
**Решение:** Переформулируйте вопрос, используйте ключевые слова из документа

### ❌ "AI путает данные из разных файлов"
**Решение:** Деактивируйте ненужные источники, оставьте только нужный

### ❌ "Excel Analyzer показывает не все строки"
**Причина:** Лимит контекста GPT (~500 строк)
**Решение:** Используйте фильтры в вопросе: "Покажи только категорию X"

## Советы для лучших результатов

### 1. Называйте файлы понятно
```
❌ file_v2_final_NEW.xlsx
✅ Каталог_продуктов_2024.xlsx
```

### 2. Используйте заголовки в Excel
```
❌ | A | B | C |
   | Кабель | 500 | шт |

✅ | Название | Цена | Единица |
   | Кабель   | 500  | шт      |
```

### 3. Задавайте конкретные вопросы
```
❌ "Расскажи про кабели"
✅ "Покажи все оптические кабели с ценой от 500 до 1000"
```

### 4. Указывайте источник если нужно
```
"Согласно файлу Прайс_лист_2024.pdf, какая цена на..."
```

### 5. Активируйте только нужные источники
Меньше активных источников = точнее ответы

## Резюме

```
📁 Загрузка → 🔪 Разбивка на части → 🔢 Создание "отпечатков"
                                              ↓
❓ Вопрос → 🔍 Поиск похожих частей → 🤖 GPT формирует ответ → 💬 Ответ
```

**Ключевые моменты:**
1. AI работает только с **активными** источниками
2. Документы разбиваются на **части** (chunks)
3. Поиск идёт по **смыслу**, не по словам
4. **Проектные** источники приоритетнее **глобальных**
5. Для детального анализа Excel используйте **Excel Analyzer**
